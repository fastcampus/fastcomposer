fastcomposer
============

a layout based WYSIWYG html editor.

## roject setup

```
npm install
```

### Compiles and generates commonjs/umd bundle

```
npm run bundle
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```
